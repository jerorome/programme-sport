# APEX — Programme Sport

**URL:** https://programme-sport.vercel.app
**Repo:** github.com/jerorome/programme-sport
**Status:** ✅ Live

## Stack
Next.js 15 + React 19 + Supabase + Tailwind CSS + Vercel

## Auth
Google OAuth + Magic Link via Supabase

## Features
- Suivi de séances de musculation
- Programmes personnalisés (V-Taper, etc.)
- Suivi de progression des charges
- Interface PWA mobile-friendly
